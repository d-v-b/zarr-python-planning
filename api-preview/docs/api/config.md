# Configuration

The namespaced configuration substrate that replaces `donfig`: namespaced keys, environment overrides (`ZARR_*`), and presets. Also hosts the public `to_sync` bridge that replaces `zarr.core.sync.sync()`. Source: [missing-apis.md §6](../proposals/missing-apis.md), [performance.md](../proposals/performance.md).
