# Codec ABCs (removed in 4.0)

The async, batched `BaseCodec` family (`ArrayArrayCodec` / `ArrayBytesCodec` / `BytesBytesCodec`), the partial-decode/encode mixins, and `CodecPipeline`. **Replaced by** the sync-first [`zarr.codec.Codec`](../api/codec.md) capability bundle. Source: [codecs.md](../proposals/codecs.md).
